# Introduction

### 阅读方式

**细读**: 都要读懂, 必须都理解, 反复读, 找额外资料继续阅读

**通读**: 大致了解一下可以干什么, 尽量读懂

**粗读**: 随手翻阅, 读不懂可以跳过, 时不时回头看
